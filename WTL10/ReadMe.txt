安装方法：
1.拷贝WTL到X:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\VCWizards\AppWiz目录下
2.拷贝Include下的所有文件到X:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\include目录下
3.管理员运行CMD命令行,运行Setup.js
  >Setup.js
4.安装完成
